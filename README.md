# Portfolio - Styzz

Ce dépôt contient un site portfolio moderne, responsive, et entièrement en français pour Styzz.

Comment utiliser

- Ouvrez `index.html` dans un navigateur pour prévisualiser le site.
- Ajoutez vos images pour les vignettes dans `assets/` en utilisant les noms `1.png`..`5.png` (les aperçus et la modal les utilisent via `data-img`).
- Les boutons "Voir" ouvrent une version agrandie (responsive) de la création dans une modal.
- Contact: `styzzpro@gmail.com` et Discord `styz_.` (copie via bouton).
- Les services sont affichés comme gratuits. Il n'y a pas de page de prix ni de formulaire sur le site.

Plus de créations

- https://www.behance.net/styzzstyzz1

Structure

- `index.html` — Page principale (header, hero, à propos, créations, contact, footer)
- `styles.css` — Feuille de styles principale (responsive)
- `script.js` — Interactions (menu mobile, modal, copy Discord/email, toast)
- `assets/` — Images et SVG (ajoutez `1.png`..`5.png` pour vos créations)

Notes

- Si vous souhaitez un formulaire côté serveur pour récupérer les demandes, je peux ajouter Formspree / Netlify / EmailJS.
- Pour un rendu optimal, fournissez des images carrées pour les vignettes et des images plus grandes pour la modal (jpg/png).
